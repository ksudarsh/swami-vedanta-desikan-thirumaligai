# Swami Vedanta Desikan Thirumaligai, Srirangam

This repository contains the working material for a donor-facing custom HTML page for `parakalamatham.org`.

## Delivery Mode
- WordPress page template: `Unboxed Content`
- Deliverable: custom HTML fragment
- Not a full standalone website

## Current Working Files
- `Requirements.md` - living project brief
- `page.html` - first working donor-page draft to be refined in later iterations
- `principal-information-template.md` - editable questionnaire text
- `principal-information-template.docx` - Word version to send to principals / trustees
- source images and project letter scans in this folder

## Current Project Focus
- establish the historical and sampradaya importance of the site
- connect it clearly to Parakala Matham
- explain the current restoration need
- prepare for future contribution details on the same page

## Important Note
This project is expected to go through multiple iterations. Do not treat the current HTML as final until the principals and trustees provide missing information and approve the historical / donor wording.
